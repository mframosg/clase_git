# Mi Calculadora V1.0

Una calculadora interactiva de línea de comandos desarrollada en Python. Simple, intuitiva y con manejo de errores integrado.

---

## Características

- **Suma** — Suma dos números
- **Resta** — Resta dos números
- **Multiplicación** — Multiplica dos números
- **División** — Divide dos números con protección contra división por cero
- **Validación de entradas** — Manejo de errores para entradas no numéricas
- **Interfaz interactiva** — Menú en consola fácil de usar

---

## Requisitos

- Python 3.6 o superior

No requiere dependencias externas.

---

## Instalación

```bash
git clone <url-del-repositorio>
cd clase_git
```

---

## Uso

```bash
python hola.py
```

Al ejecutar el programa verás el siguiente menú:

```
=== Mi Calculadora V1.0 ===

Operaciones disponibles:
1. Sumar
2. Restar
3. Multiplicar
4. Dividir
5. Salir

Selecciona una operación (1-5):
```

Selecciona el número de la operación deseada, ingresa los dos números y obtén el resultado al instante.

### Ejemplo de uso

```
Selecciona una operación (1-5): 1
Ingresa el primer número: 10
Ingresa el segundo número: 5
Resultado: 15.0
```

## Manejo de errores

| Situación | Respuesta |
|---|---|
| División por cero | `Error: No se puede dividir por cero` |
| Entrada no numérica | `Error: Debes ingresar números válidos` |
| Opción de menú inválida | `Opción no válida` |
